types of css

inline css

Inline CSS is a method of adding styles directly to an HTML element using the style attribute. This allows you to change the appearance of a single element quickly without needing an external stylesheet. For example, you can set the color or font size of a paragraph inline. Although inline CSS is easy to use for small changes or quick tests, it is not efficient for larger projects. It mixes style with content, making the code harder to read and maintain. It also doesnâ€™t allow reusing styles across multiple elements. For these reasons, inline CSS is best used sparingly.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8" /> <title>Inline CSS Example</title> </head> <body> <p style="color: green; font-weight: bold"> This text is styled using inline CSS. </p> </body> </html>

internal css

Internal CSS refers to placing CSS styles inside an HTML document within the style tag, usually in the head section. This method allows you to apply styles to the entire page without linking to an external stylesheet. Internal CSS is useful when you want to style a single page uniquely or test styles quickly. While internal CSS keeps the styles within the HTML file, making it easier to manage small projects, it is less efficient for larger websites where many pages share the same styles. For bigger projects, external CSS files are recommended for better organization and reusability.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8" /> <title>Internal CSS Example</title> <style> body { background-color: #f0f8ff; font-family: Arial, sans-serif; } h1 { color: navy; text-align: center; } p { color: #555; font-size: 16px; } </style> </head> <body> <h1>Welcome to My Website</h1> <p>This paragraph is styled using internal CSS.</p> </body> </html>

external css

External CSS involves writing all your CSS styles in a separate .css file, which you then link to your HTML document using the tag. This approach is ideal for larger websites because it allows you to keep your styles organized and reuse them across multiple pages. Using external CSS improves maintainability, as any change in the CSS file instantly updates the look of all linked pages. It also keeps the HTML files clean and focused on content rather than styling. External CSS is widely used in professional web development for its scalability and separation of concerns.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8" /> <title>External CSS Example</title> <link rel="stylesheet" href="styles.css" /> </head> <body> <h1>Welcome to My Website</h1> <p>This paragraph is styled using external CSS.</p> </body> </html>
